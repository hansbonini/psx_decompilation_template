# PSX Decompilation Template
A template for PSX decompilation projects.

## About
This is intended to make it quick and easy to set up and start a PSX decompilation project.

# How to use
- Place your game ISO inside the `iso` directory and rename `.cue` file to `SLUS-XXXXX.cue`.
- Run `make setup`
